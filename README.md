#伝説の超！！バッティングセンター

## 概要

C++ / DxLib で制作した 2D超次元バッティングゲームです。
「遊んだあとに誰かに話したくなる体験」を設計の軸として開発しました。

---

## 動作環境

OS : Windows 10 / 11（64bit）

解像度 : 1280 × 720

フレームレート : 60FPS 

推奨コントローラー : キーボード

---

## 開発環境

言語 : C++ 

ライブラリ : DxLib 

IDE : Visual Studio

---
## ゲーム内容

特徴のあるキャラを選んでゲームスタート！
パワーを溜めてタイミングよくスイング！！
ホームランを狙おう！！


めざせコレクションコンプリート！

---

## よくありそうなご質問


Q.ホームラン出ないんだけど？

ホームランが出るには条件があります。

パワーゲージが赤の時でかつ、特定の確率で出ます。
